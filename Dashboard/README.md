# Augmedix Data Operations - PT Reconciliation Dashboard

This repository contains the interactive React dashboard built to address **Problem 2 (Closed Encounters Analysis)** of the Central Operations Case Study.

## Architecture
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Visualizations:** Recharts
- **Icons:** Lucide-React

## Features
- **Executive Overview:** High-level metrics tracking the 94.16% match rate across 26,716 CPT lines.
- **Provider Performance:** Detailed RAG (Red/Amber/Green) status for individual clinical providers.
- **CPT Code Analysis:** DB vs EHR discrepancies filtered down to the specific CPT code.
- **Gap & Operations:** Centralized action plan targeting the 1,357 EHR-only and 202 DB-only lines.
- **Day/Night Mode:** Fully responsive and accessible color themes.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## Data Source
The data powering this dashboard (`src/data.ts`) was generated by running a custom Python/SQLite ETL script (`SQL/load_and_analyze.py`) against the provided raw CSV datasets.
